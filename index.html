Examples

Keyboard shortcuts

  • B
    Fill bucket
  • C
    Choose color
  • P
    Pencil
  • L
    Stroke
  • E
    Eraser
  • A
    Paint all pixels of the same color
  • G
    Save as .gif
  • Y
    Save as .apng
  • F
    Full screen
  • N
    Add new frame
  • R
    Delete current frame
  • D
    Duplicate current frame
